# Vi Editor
`vi` text editor tips.

## Common Tips
```
CTRL-V    u                  Change case
SHIFT-I   spaces             Indent
INDENT BLOCK
  Go to line, and press v then jj, selecting all the lines you want
  :le 4   for 4 spaces
UNIDENT ENTIRE FILE
  :gg=G
SET AUTOINDENT
  :set ai
SEARCH/REPLACE g multiple instances per line, c confirm
  :%s/OLD/NEW/gc             
```
